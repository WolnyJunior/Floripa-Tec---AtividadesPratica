Vamos começar o desenvolvimento de uma calculadora para aplicar o conhecimento adquirido durante o curso. Crie um documento HTML com a seguinte estrutura:

  1. Um título h1 com o nome da sua calculadora
  2. Um pequeno parágrafo descrevendo seu projeto (use sua criatividade!)
  3. Uma lista ordenada com o passo-a-passo para usar a sua calculadora
  4. Dois elementos input do tipo número
  5. Um elemento select com as opções sendo as principais operações matemáticas
  6. Um botão para realizar o cálculo
  7. Um parágrafo com os dizeres “O resultado da operação é:”. Posteriormente usaremos JavaScript para inserir o valor neste parágrafo.
  8. Obs. Não se preocupe com estilos e funcionalidades neste momento.
